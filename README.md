# 🎬 Moview

A lightweight movie discovery application built with **HTML**, **CSS**, and **Vanilla JavaScript**. Browse popular movies on load, search by title, and view ratings with hover-over overviews — all powered by the **TMDB API**.

## 🚀 Features

- 🎥 **Popular Movies** – Displays trending movies sorted by popularity on page load.
- 🔍 **Movie Search** – Search movies by title using the TMDB search endpoint.
- ⭐ **Rating Badges** – Color-coded ratings: green (≥ 8), orange (≥ 5), red (< 5).
- 📋 **Hover Overview** – Slide-up overlay with movie title and description on hover.
- 🌙 **Dark Theme** – Clean dark UI with Space Grotesk typography.
- 📱 **Responsive Grid** – Flexible movie card layout that wraps across screen sizes.
